### 步骤1 克隆代码到本地
```
# git clone git@github.com:15129230785/gitbook.git
# cd gitbook
```

### 步骤2 基于主干拉个人开发分支
```
# git checkout -b develop_工号

```

### 步骤3 开发代码，提交到本地，推送到远程仓库
```
# git push origin develop_工号

```

### 步骤4 在git网站上提MR
![img](/img/MR.png)

### 步骤5 Committer合入
> [!WARNING|style:flat]
